# escape-game

>利用HTML+CSS+JavaScript写的一个简单游戏。

<br>

包含的内容有：
- 隐藏图片鼠标悬浮显示
- 点击图片/文字跳转其他页面
- 打字机效果文本显示
- 井字棋
- 打字游戏
  - 增加了匹配正确时进行字体和颜色随机变换效果
  - 匹配错误时红色背景标红

<br>
其中的井字棋游戏直接参考的<a href="https://github.com/SudeepAcharjee/The-50-Front-end-Project/tree/main/12.Tic-tac-toe%20Game">Tic-tac-toe Game</a><br>
